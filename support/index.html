Support and contact

If something is not working, start here. This page is for course support and signposting.

Course support first

  • If a page looks odd, try refreshing (Ctrl + F5).
  • If you are on a phone, try rotating your screen or using a larger device for the carousel.
  • If you need a simple fallback, use Downloads for print-ready materials.

If you are properly stuck

You can contact Tidy Butt directly using their contact page: tidybutt.co.uk/pages/contact

Get help now

If you need urgent help, use the Tidy Butt Get Help page: tidybutt.co.uk/pages/get-help

You are not expected to do everything perfectly. If your energy is low, focus on the Get Ready steps for your next session.
