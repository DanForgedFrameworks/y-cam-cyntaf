Support and contact

This page is here to help you keep going with the course. If something is getting in the way, you are not on your own.

If you are in immediate danger, call 999.

If you are stuck with the course

Try one of these (pick the easiest one):

  • Re-open the Orientation page and check the “how it works” bit.
  • Use Downloads if you prefer print materials or offline access.
  • Bring one question to the next session. Even one is enough.
Go to Orientation

If you need to talk to someone

If you need support beyond the course, Tidy Butt can help point you in the right direction.

If you are properly stuck: use the contact page.

Urgent help

If you are in crisis or need immediate support, contact one of these trusted helplines. Professional help is available 24/7.

Samaritans — Call 116 123

Papyrus Hopeline UK — Call 0800 068 4141 | Text 07786 209 697

Mind — Call 0300 123 3393 | Text 86463
